## 🎒 生徒のみなさんへ

### はじめてのとき(1回だけ)

1. 運営から Discord の DM で**招待リンク**が届きます
2. リンクを開いて「**Discord でログイン**」を押す(いつもの Discord アカウントで OK)
3. 表示名(ニックネーム可)を入れて、説明を読んで同意にチェック → 登録完了!

⚠ 招待リンクは **72時間で期限切れ**になります。切れていたら運営に再発行を頼んでください。

### 毎日やること(30秒)

- アプリを開くと「**今週の課題**」が出ます。やることはここを見れば全部わかります
- 課題に手をつけたら「**進行中**」、終わったら「**完了**」をタップ。**それだけ**です
- 「p.22 まで終わった」みたいなメモを残したいときは、課題を開いてメモ欄へ

### テストが返ってきたら

- 「**成績登録**」から、テスト名・科目・点数を入力(2週間以内に登録するのが目標!)
- 「**マイ成績**」で自分の成績の伸びがグラフで見られます。**他の人からは絶対に見えない**ので安心してください

### 困ったら

質問・相談は今までどおり **Discord** で。アプリは「記録する場所」、会話は Discord です。

## 👩‍🏫 講師のみなさんへ

- **ダッシュボード**: 担当生徒 × 課題の表で、誰がどこまで進んだか色で一望できます。DM での個別確認は不要になります
- **課題作成**: タイトル・締切・割当先(担当生徒にまとめて割当可)を入れて作成。作成すると Discord に自動通知が飛びます。毎週同じ形式なら、ダッシュボードの「**複製**」が便利です
- **成績登録**: 小テストの採点をしたら、その場で担当生徒の成績を登録してください(模試・定期は生徒の自己申告が基本、代理登録も可)。**分野別の内訳**(例: 数学>二次関数)を付けておくと、将来の学習提案の材料になります
- **生徒詳細**: 生徒名をタップすると、課題履歴と成績推移を縦断的に見られます
- 見られるのは**現在担当している生徒のデータだけ**です。担当の追加・変更は運営に依頼してください

## 🛠 運営のみなさんへ

日常操作はすべて `/admin` 画面から:

| やること | 操作 |
|---|---|
| メンバー追加 | 招待リンクを発行(生徒の場合は担当講師も指定)→ Discord DM で本人へ |
| 担当の設定・変更 | 「担当関係」で講師⇄生徒を設定・終了 |
| 退会・利用停止 | ユーザー一覧から「無効化」 |
| 全体予定の登録 | 「予定を登録」(模試日・オフライン会など。特定生徒だけに見せることも可) |

セットアップ・KPI 集計・データ削除など詳細は [deployment.md](deployment.md) を参照。
新規生徒は**保護者の同意確認 → 招待発行**の順を守ってください([consent-form.md](consent-form.md))。
