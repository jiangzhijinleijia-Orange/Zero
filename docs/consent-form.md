# 保護者同意フォーム — 作成キット

> **これは何か**:保護者の方から「アプリ利用への同意」をいただくための **Google フォームを、迷わず作れるようにした完成キット**です。
> **なぜ手作業か**:Google フォームは *あなたの Google アカウントの中* に作るものなので、こちらで直接は作れません。その代わり、中身は全部用意しました。あなたは **「貼って、並べる」だけ** です(所要 10〜15分)。
> **あなたが埋める空欄は、たった1か所**:運営用メールアドレス(後述の `【メール】`)。これだけです。

---

## 0. 全体像 — 何を作るのか

作るのは **1枚の Google フォーム**です。保護者がそれを開き、説明を読み、お子さまの名前などを記入し、最後に「同意します」にチェックして送信する——その送信記録が「同意の証拠」になります。

流れはこうつながっています:

```
保護者がこのフォームで同意  →  運営が招待リンクを発行  →  生徒がアプリに登録(本人も同意)
```

**順番が大切**です。必ず「**保護者の同意 → 招待**」の順で。生徒を招待する前に、保護者の同意を受け取っておきます。

---

## 1. フォームに載せる「説明文」(まるごとコピペ)

下の枠の中を**まるごとコピー**して、Google フォームの**説明欄**(タイトルのすぐ下)に貼り付けてください。
`【メール】` の部分だけ、あなたの運営用メールアドレスに置き換えます。

```
蒼理塾(琉球サーティー)では、お子さまの学習をサポートするために、無料の学習管理アプリを使います。お子さまの情報を一部お預かりするため、内容をご確認のうえ、ご同意をお願いいたします。

■ このアプリですること
・講師が出した課題(宿題)の確認と、進み具合の記録
・模試やテストの結果の記録と、成績の推移をグラフで見えるようにすること
・模試の日程やオフライン会などの予定の共有

■ お預かりする情報 / お預かりしない情報
お預かりするのは、表示名(ニックネームで構いません)・学年・課題の進み具合・テストの点数や偏差値などです。
本名・住所・電話番号・写真などは、お預かりしません。

■ お子さまの成績を見られる人
お子さま本人・担当の講師・蒼理塾の運営だけです。ほかの生徒からは見えません。これはお願いやマナーではなく、システムの仕組みとして守られており、万一アプリに不具合があっても、ほかの生徒に見えることはありません。成績を外部に公開したり、第三者へ渡したりすることもありません。

■ 安全管理について
情報は暗号化された通信でやりとりされ、信頼できるクラウドサービス上に保管されます。蒼理塾は無償の学習支援コミュニティであり、お預かりした情報を営利目的で使うことはありません。

■ 費用
無料です。アプリの利用にお金がかかることは一切ありません。

■ やめたいとき
いつでも利用をやめられます。下記のお問い合わせ先までご連絡いただければ、お子さまのデータ(成績や課題の記録など)を、原則1か月以内にすべて削除します。

■ お問い合わせ先
ご質問やご不安な点は、下記までお気軽にご連絡ください。
蒼理塾 運営  メール:【メール】

下の項目にご記入のうえ、いちばん下の「同意します」にチェックを入れて送信してください。
```

**フォームのタイトル**(説明欄の上)は、これにします:

```
蒼理塾 学習管理アプリ 利用同意のお願い(保護者の方へ)
```

## 2. 質問の一覧(この6つを、この順番で作る)

説明文の下に、次の6つの質問を並べます。「種類」は Google フォームで選ぶ回答形式、「必須」はオンにするものです。

| # | 質問文(そのままコピー可) | 種類 | 必須 | 選択肢 |
|---|---|---|---|---|
| 1 | 生徒のお名前(アプリで使うニックネームでも構いません) | 記述式(短文) | ✔ | — |
| 2 | 生徒の学年 | プルダウン | ✔ | 中1 / 中2 / 中3 / 高1 / 高2 / 高3 / その他 |
| 3 | 保護者のお名前 | 記述式(短文) | ✔ | — |
| 4 | お子さまとの続柄 | ラジオボタン | ✔ | 父 / 母 / その他 |
| 5 | 上記の説明を読み、お子さまのアプリ利用に同意します | チェックボックス | ✔ | 同意します |
| 6 | ご質問・ご要望があればご記入ください(任意) | 記述式(段落) | — | — |

> **ポイント**:質問5の「同意します」を**必須のチェックボックス**にしておくと、チェックを入れないと送信できません。これが「同意の関門」になります。
> **集めすぎない**のが安全です。住所・電話番号などは集めません(アプリ本体と同じ、最小限の情報だけ)。

## 3. Google フォームでの作り方(順番どおりに)

1. パソコンかスマホで [forms.google.com](https://forms.google.com) を開く(Google アカウントでログイン)
2. **「空白のフォーム」**(＋の新規作成)を選ぶ
3. いちばん上の **「無題のフォーム」** をタップして、§1 の**タイトル**を入力
4. そのすぐ下の **「フォームの説明」** 欄に、§1 の**説明文をまるごと貼り付け**、`【メール】` を運営用メールアドレスに直す
5. 質問を1つずつ追加する。右側(スマホは下)の **「＋」ボタン**で質問を増やし、§2 の表のとおりに——
   - **質問文**をコピーして貼る
   - **種類**(記述式・プルダウン など)を選ぶ
   - 選択肢があるものは、表のとおり1つずつ入力する
   - 表で「✔」のものは、その質問の右下の **「必須」をオン**にする
6. 6つすべて作り終えたら、入力ミスがないか上から一度見直す

これで本体は完成です。

---

## 4. 仕上げの設定(任意だが、やると安全)

右上の **歯車(設定)** から、次を確認しておくと安心です:

- **「メールアドレスを収集する」は オフ**のまま(保護者の Google アドレスまで集めない=最小限に)
- **「回答を1回に制限」は オフ**でOK(保護者がログインせず使えるように)

回答の保存先:画面上部の **「回答」タブ → スプレッドシートのアイコン**で、回答一覧表(Google スプレッドシート)を作れます。
この表には生徒名・保護者名が入るので、**共有はしない**でください(運営の同意管理担当だけが見る)。送信された日時が、そのまま「いつ同意をもらったか」の記録になります。

---

## 5. 保護者への配り方

1. 右上の **「送信」** → **リンク(🔗)**を選び、URL をコピー(短縮にチェックすると短くなります)
2. その URL を保護者に届ける。届け方は2通り:
   - **生徒経由**:生徒に「おうちの人に渡してね」とリンクを伝える(いちばん手軽)
   - **直接**:把握している連絡手段があればそちらへ
3. 保護者が記入・送信 → 運営が回答(スプレッドシート)で確認
4. **同意を確認できたら、その生徒に招待リンクを発行**(`/admin` から)

---

## 6. 運用ルール(これだけ守る)

- **順番**:保護者の同意 → 招待リンク発行 → 生徒がアプリ登録。これを必ず守る
- **二段構え**:保護者はこのフォームで、生徒本人はアプリの初回登録画面で、それぞれ同意する仕組みになっている
- **記録の保管**:回答スプレッドシートは運営の同意管理担当が管理し、外部に共有しない
- **削除依頼が来たら**:お問い合わせ先(運営メール)で受け、対象の生徒データをアプリ側で削除(手順は [deployment.md](deployment.md) の「退会(データ削除)」)。原則1か月以内
